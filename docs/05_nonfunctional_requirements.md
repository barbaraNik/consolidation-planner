### Нефункциональные требования
| Категория | Требование |
|---|---|
| Производительность | Ответ API на расчёт загрузки (до 50 заказов) — не более 3 секунд. |
| Производительность | Список заказов (до 500 записей) — не более 1 секунды. |
| Надёжность | Все операции записи выполняются в транзакции: либо все операции выполняются успешно, либо при ошибке происходит полный откат. |
| Безопасность | API защищён JWT-авторизацией, HTTPS. |
| Совместимость | Веб-интерфейс работает в браузерах на базе Chromium. |
| Масштабируемость | Система должна поддерживать одновременную работу не менее 10 пользователей без деградации. |
